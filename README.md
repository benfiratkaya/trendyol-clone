# Proje Kurulumu

Proje React.js kullanılarak yapılmıştır. Paket yöneticisi olarak yarn kullanılmaktadır.

### Kurulum:

```bash
yarn
```

### Geliştirme:

```bash
yarn start
```

### Build:

```bash
yarn build
```
